====================
1-my_list.py testing
====================


Testing MyList class and its print_sorted() method


Setup
=====
    >>> MyList = __import__("1-my_list").MyList


Check that MyList is a list subclass
====================================
    >>> issubclass(MyList, list)
    True

Empty list check
================
	>>> l = MyList()
	>>> l.print_sorted()
	[]

append() check
==============
	>>> l.append(1)
	>>> l.append(2)
	>>> l.append(3)
	>>> print(l)
	[1, 2, 3]

ordered list check
==================
	>>> l.print_sorted()
	[1, 2, 3]

reverse ordered list + negative num check
=========================================
	>>> l = MyList()
	>>> l.append(3)
	>>> l.append(2)
	>>> l.append(1)
	>>> l.append(-5)
	>>> print(l)
	[3, 2, 1, -5]
	>>> l.print_sorted()
	[-5, 1, 2, 3]
	>>> print(l)
	[3, 2, 1, -5]

all negative numbers check
==========================
	>>> l = MyList()
	>>> l.append(-2)
	>>> l.append(-8)
	>>> l.append(-4)
	>>> print(l)
	[-2, -8, -4]
	>>> l.print_sorted()
	[-8, -4, -2]
	>>> print(l)
	[-2, -8, -4]
